✏️ Air Doodle

Draw in the air with your finger. No touch, no stylus, just magic.

🖐️ Hand Tracking

Advanced AI-powered hand tracking. Just extend your index finger and draw in mid-air.

🎨 Smart Brushes

Solid lines, spray paint, rainbow colors, and symmetry mode for creative effects.

🎬 Record & Replay

Record your drawing sessions and play them back with our built-in player.

⌨️ Full Control

Undo/redo, shapes, backgrounds, and more - all controlled via simple keyboard shortcuts.

🚀 Quick Start

# Clone the repository git clone https://github.com/pravallikakolagani/air-doodle.git cd air-doodle # Install dependencies pip install -r requirements.txt # Run Air Doodle python air_doodle.py

🎮 Controls

Index Finger - Draw
c - Change color
+ / - - Stroke width
e - Eraser mode
b - Background
v - Brush type
m - Symmetry
n - Rainbow mode
r / t - Recording / Timelapse
z / y - Undo / Redo
1/2/3 - Shapes
s / l - Save / Load
